# Xiaoyu Zhang - Personal Website

This is my personal website hosted on GitHub Pages, showcasing my coursework and projects in data science and statistics.

## Website Structure

- **Homepage** (`index.html`): Introduction, skills, and project overview
- **Q2 Report** (`blog.html`): Detailed coursework report on survival analysis and Text-to-SQL experiments
- **Styling** (`style.css`): Custom CSS for responsive design

## Projects Featured

### Q2 Survival Analysis Reproduction
- Reproduction of Databricks survival-analysis workflow
- PySpark data processing with Bronze/Silver table construction
- Kaplan-Meier, Cox Proportional Hazards, and Accelerated Failure Time models
- Customer Lifetime Value (CLV) estimation

### Text-to-SQL Failure Analysis
- Evaluation of GPT-based SQL generation
- Analysis of failure cases: ambiguity, schema grounding, dirty data, syntax errors
- Successful case study with correct SQL generation

## Technologies Used

- **Frontend**: HTML5, CSS3 (responsive design)
- **Data Analysis**: Python, PySpark, pandas, lifelines
- **Database**: MySQL for Text-to-SQL experiments
- **Hosting**: GitHub Pages
- **Version Control**: Git
